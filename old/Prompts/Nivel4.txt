Seu objetivo é projetar uma plataforma completa de suporte remoto corporativo baseada em RustDesk Open Source, mas com recursos avançados de gestão, controle e produto, semelhante a um SaaS profissional.

MISSÃO

Projetar uma plataforma open source de suporte remoto corporativo, que permita que empresas ofereçam suporte técnico remoto para seus clientes com:

gestão centralizada

multi-tenant

controle de acesso

auditoria completa

segurança corporativa

Essa plataforma deve funcionar como um AnyDesk/TeamViewer self-hosted empresarial.

PRINCÍPIOS DO PROJETO

A arquitetura deve seguir estes princípios:

✔ Open Source
✔ Self-hosted
✔ Segurança Zero Trust
✔ Escalabilidade horizontal
✔ Infraestrutura automatizada
✔ Observabilidade completa
✔ Multi-tenant
✔ Alta disponibilidade

CONTEXTO DO PRODUTO

A plataforma será usada por:

empresas de suporte técnico

MSPs (Managed Service Providers)

equipes internas de TI

empresas de ERP

Escala esperada:

10–100 técnicos

500–5000 dispositivos remotos

milhares de sessões mensais

RESTRIÇÕES

Não utilizar:

RustDesk Server Pro

SaaS proprietários

ferramentas pagas

Utilizar apenas:

RustDesk Open Source

tecnologias open source

infraestrutura self-hosted
