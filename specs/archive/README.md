# Arquivo Histórico de Engenharia

Este diretório contém documentos legados e materiais de referência que foram utilizados na fase de concepção e design inicial da plataforma.

## Conteúdo

### 📁 [prompts/](prompts/)
Contém os prompts hierárquicos utilizados para "instruir" o comitê de engenharia virtual durante o Spec-Driven Development. 
- **Prompt final.txt**: O documento mestre de requisitos que definiu a arquitetura 1.0.
- **Nivel1.txt a Nivel6.txt**: Logs de iterações anteriores do design.

### 📄 [RustDesk-Platform-Blueprint.docx](RustDesk-Platform-Blueprint.docx)
Documento Word original com o blueprint inicial do projeto.

---
> [!NOTE]
> Estes arquivos são mantidos apenas para fins de auditoria histórica. Para desenvolvimento atual, utilize as especificações em `specs/` e os guias em `docs/`.
